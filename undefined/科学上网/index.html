Windows、Android、IOS科学上网喂饭教程

当我们有需求访问国外的网站(例如购物、学术论文、视频平台、Github、Google、Ai···)时会出现DNS无法解析的情况，导致网站一直转圈圈无法加载，这时候我们就需要代理工具和来帮助我们，以下是喂饭教程，如果实在学不会请放弃另想办法，下面我会带大家认识代理协议和代理工具以及如何安装和使用

代理协议

​ 根据社区活跃度和工具支持情况，目前大家常用的当前主流的代理协议大致如下：

常见代理工具

Clash系：Clash 是一个基于规则的跨平台代理工具核心，支持多种协议（如 Shadowsocks、V2Ray、Trojan、Socks5 等)

V2Ray系列：V2Ray 是一个功能强大的代理工具核心，支持 VMess（自研协议）、VLESS、Shadowsocks 等协议。它的 Windows 客户端以 V2RayN 为代表，因其稳定性和灵活性受到技术爱好者喜爱。

V2RayN: 最经典的 V2Ray Windows GUI 客户端。
特点：UI 清晰，功能菜单逻辑性强，支持手动配置和订阅导入，扩展性高。
V2RayNG:原本是 Android 端的 V2Ray 客户端，但也有 Windows 版本（V2RayNG-Windows）。
        当前状态：不如 V2RayN 在 Windows 上流行，但部分用户仍在使用。
        特点：轻量，适合熟悉 V2RayNG Android 版的用户跨平台使用。

其他系列：除了 Clash 和 V2Ray，还有一些独立的代理工具或新兴系列也有一定用户基础

sing-box: 一个较新的代理工具核心，对协议支持极为丰富（包括 Shadowsocks、VLESS、Hysteria 等），性能据称优于 V2Ray 和 Clash

Surge: 一款商业化的高端代理工具，支持多种协议（SS、Trojan、V2Ray 等）和规则分流。功能极其强大，支持 macOS 原生特性（如网络扩展）

Quantumult X:一款商业代理工具，原为 iOS 设计，现支持 macOS。支持复杂规则和多种协议，界面优雅。

Stash:基于规则的代理客户端，主要面向苹果生态（iOS、macOS、tvOS 等），完全兼容 Clash Premium 的配置。它支持多种代理协议（如 VLESS、Trojan、Hysteria、Shadowsocks 等），并提供强大的规则分流功能。

Loon:一个功能强大的代理工具，最初为 iOS 设计，后来也支持 macOS。它支持多种协议（Shadowsocks、V2Ray、Trojan 等），以其简洁的界面和灵活的配置著称。支持规则分流和策略组，配置方式类似 Surge,提供插件系统，可以扩展功能。

Karing:一个相对较新的代理工具，支持 Clash 和 sing-box 的路由规则，跨平台支持包括 macOS、iOS、Windows 和 Android。简单易用和强大功能,完全免费支持多种协议（Shadowsocks、VLESS、Hysteria 2、WireGuard 等）

Windows和Android可以直接在浏览器访问Github下载，IOS需要在APP Store更换外国ID下载
